# Text Annotation App

Приложение для аннотирования текста на Angular 18.

## Возможности

- **CRUD для статей**: создание, редактирование, удаление, просмотр
- **Аннотирование текста**: выделение фрагментов, выбор цвета, текстовые примечания
- **Всплывающие подсказки** при наведении на аннотированный текст
- **Сохранение в localStorage** (все данные сохраняются после перезагрузки)

## Технологии

- Angular 18
- TypeScript
- RxJS
- Range API (выделение текста)
- LocalStorage API
